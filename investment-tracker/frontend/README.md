# Investment Tracker Frontend

This is the frontend part of the Investment Tracker project, built using React. The application allows users to manage their investments in different ETFs and assets, providing a user-friendly interface for tracking changes over time.

## Getting Started

To get started with the frontend application, follow these steps:

1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd investment-tracker/frontend
   ```

2. **Install Dependencies**
   Make sure you have Node.js installed. Then, run the following command to install the necessary dependencies:
   ```bash
   npm install
   ```

3. **Run the Application**
   Start the development server with:
   ```bash
   npm start
   ```
   The application will be available at `http://localhost:3000`.

## Features

- **Manage Assets**: Create, read, update, and delete investment assets.
- **Manage ETFs**: Perform CRUD operations on exchange-traded funds.
- **Investment History**: Track and display the history of investments over time.

## Folder Structure

- `src/components`: Contains React components for managing assets and ETFs.
- `src/App.js`: Main application component that sets up routing and layout.
- `src/index.js`: Entry point of the React application.
